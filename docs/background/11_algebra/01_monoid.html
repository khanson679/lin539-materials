Semigroups, monoids, and groups

Remember that a magma is a (total) algebra whose carrier set is closed under the algebra’s operation \(\oplus\). By adding on additional properties that \(\oplus\) must satisfy, one gets a hierarchy of increasingly specialized types of algebras:

Property Algebra type
\(\oplus\) total total algebra
previous & closure under \(\oplus\) magma
previous & associativity of \(\oplus\) semigroup
previous & identity element for \(\oplus\) monoid
previous & inverse for \(\oplus\) group
previous & commutativity of \(\oplus\) Abelian group

Semigroups

The next step up from a magma is a semigroup, which requires \(\oplus\) to be associative.

A (total) magma \(\left \langle C, \oplus \right \rangle\) is a semigroup iff \(\oplus\) is associative.

Remember that associativity means \(a \oplus (b \oplus c) = (a \oplus b) \oplus c\). So order of evaluation does not matter. This is a tremendous advantage. Associativity can greatly simplify proofs, and it also allows for faster computational implementations (see, for instance, the unit on matrix multiplication).

The magma \(\left \langle \mathbb{N}, + \right \rangle\) is a semigroup because addition is associative.

The total algebra \(\left \langle \mathbb{Z}, - \right \rangle\) is the set of integers with subtraction. This is a magma but not a semigroup because \(5 - (3 - 2) = 4 \neq 0 = (5 - 3) - 2\).

For each one of the following, say whether it is a semigroup. Justify your answer.

  • the interval \([0,1]\) of real numbers between and including \(0\) and \(1\), with multiplication
  • for any given set \(S\), \(\wp(S)\) with set union (where \(\wp(S)\) is the powerset of \(S\))
  • for any given set \(s\), \(\wp(S)\) with relative complement
  • \(\Sigma^*\) with an operation splice that maps two strings \(u_1 u_2 \cdots u_m\) and \(v_1 v_2 \cdots v_n\) (\(m, n \geq 0\)) to \(u_1 v_1 u_2 v_2 \cdots u_x v_x\), where \(x \mathrel{\mathop:}=\min(m,n)\) (for instance, \(abc\) spliced with \(defghij\) is \(adbecf\), and \(abcdefg\) spliced with \(i\) is \(ai\)).

Monoid

For computational linguistics, the most important kind of algebraic structure is the monoid. A monoid is a semigroup with a unique identity element for \(\oplus\). An identity element is something that causes \(\oplus\) not to change the value of the other argument.

Addition has \(0\) as its identity element. For instance, \(5 + 0 = 5\), and similarly \(0 + 5 = 5\). So adding the identity element to some natural number \(n\) just gives us \(n\) again.

For multiplication, the identity element is \(1\) as \(5 \times 1 = 1 \times 5 = 5\).

What is the identity element for set union? Justify your answer.

A semigroup \(\left \langle C, \oplus \right \rangle\) is a monoid \(\left \langle C, \oplus, e \right \rangle\) iff there is a unique element \(e \in C\) such that for all \(a \in C\), \(a \oplus e = e \oplus a = a\). We say that \(e\) is the identity element for \(\oplus\), and that \(\oplus\) has an identity (element).

We already know that addition has the identity element \(0\), so it’s not surprising that the semigroup \(\left \langle \mathbb{N}, + \right \rangle\) is also a monoid. We can simply write \(\left \langle \mathbb{N}, +, 0 \right \rangle\) to express this fact.

The minimally different algebra \(\left \langle \mathbb{N}^+, + \right \rangle\) is a semigroup, but not a monoid. That’s because \(\mathbb{N}^+ \mathrel{\mathop:}=\left \{ 1,2,3,\ldots \right \}\) is the set of positive natural numbers, which excludes \(0\). Therefore, the algebra lacks the required identity element.

For each one of the following semigroups, say whether it is a monoid. Justify your answer.

  • \(\left \langle \wp(S), \cap \right \rangle\) (where \(S\) is some arbitrary set)
  • \(\Sigma^+\) with string concatenation
  • the set of all strings of even length with string concatenation
  • a semilattice

A special case of the monoid is the commutative monoid, where \(\oplus\) is also commutative. This means that \(a \oplus b = b \oplus a\).

The natural numbers with addition are a commutative monoid. While \(\Sigma^*\) with concatenation is a monoid (with identity element \(\varepsilon\)), it is not a commutative monoid as \(a \cdot b = ab \neq ba = b \cdot a\).

Actually, there are special cases where \(\left \langle \Sigma^*, \cdot \right \rangle\) is a commutative monoid. Describe one of these special cases.

Group

Perhaps the most studied and best understood kind of algebraic object is the group. A group is a monoid where every element also has an inverse. An element and its inverse are like polar opposites that cancel each other out to yield the identity element.

A monoid \(\left \langle C, \oplus \right \rangle\) is a group iff there for every \(a \in C\) there is an element \(a^{-1} \in C\) such that \(a \oplus a^{-1} = e\). We say that \(a^{-1}\) is the inverse of \(a\), and that \(\oplus\) has an inverse for every element of the carrier \(C\). As \(e\) is unique, we can add it directly to the specification: \(\left \langle C, \oplus, e \right \rangle\).

The monoid \(\left \langle \mathbb{N}, +, 0 \right \rangle\) is not a group. Consider any arbitrary \(n \in \mathbb{N}\). There is no natural number \(x\) such that \(n + x = 0\).

The monoid \(\left \langle \mathbb{Z}, +, 0 \right \rangle\) is a group. For every integer \(n\), its inverse with respect to addition is \(-n\). For instance, \(5 + (-5) = 5 - 5 = 0\), and \(-3 + (- (- 3)) = -3 + 3 = 0\).

While groups are mathematicians’ darling, they show up much less in linguistics and are mostly listed here for the sake of completeness. That said, there are some promising attempts to find group-like mechanisms in language, so a few years from now groups might be just as important as monoids.

For each one of the following monoids, say whether it is a group. Justify your answer.

  • \(\left \langle \mathbb{N}, \times, 1 \right \rangle\)
  • \(\left \langle \mathbb{Q}, \times, 1 \right \rangle\) (where \(\mathbb{Q}\) is the set of rationals; e.g. \(\frac{3}{2} \times \frac{-2}{2}\))
  • \(\Sigma^{\geq 3} \cup \left \{ \varepsilon \right \}\) with concatenation and identity element \(\varepsilon\) (\(\Sigma^{\geq 3}\) contains all \(\Sigma\)-strings whose length is at least \(3\))
  • \(\wp(\Sigma^*)\) with union

The monoid \(\left \langle \Sigma^*, \cdot, \varepsilon \right \rangle\) is not a group because there are no inverses. No matter what one concatenates a string \(u\) with, it won’t reduce \(u\) to \(\varepsilon\). But if one really wants the monoid to be a group, one can stipulate the existence of such inverses.

For any string \(u\) let \(\overline{u}\) be such that \(u \cdot\overline{u} = \overline{u} \cdot u = \varepsilon\). In cases where the two don’t match, nothing happens: \(u \cdot\overline{v} = \overline{v} \cdot u = u\) if \(u \neq v\). So \(\mathit{abc} \cdot\overline{\mathit{abc}} = \varepsilon\), but \(\mathit{abc} \cdot\overline{\mathit{aac}} = abc\). Intuitively, \(\overline{\mathit{abc}}\) is an instruction to delete a string iff it is \(\mathit{abc}\); basically the string equivalent of antimatter in physics.

Thanks to these “antistrings”, \(\left \langle \left \{ u, \overline{u} \mid u \in \Sigma^* \right \}, \cdot, \varepsilon \right \rangle\) is a group.

For some groups, \(\oplus\) may also be commutative, so that \(a \oplus b = b \oplus a\) for all \(a\) and \(b\). An example of that is \(\left \langle \mathbb{Z}, + \right \rangle\). Such a group is called Abelian, after the mathematician Niels Henrik Abel.
